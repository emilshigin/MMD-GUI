# MMD-GUI

## Description
### Objective:
* Get information from VR Headset Devices 
* Push files onto the headset

### Technology
* Python
* TKinter
* adb-shell[usb]

## DEMO Features
### Scan Button
Press "Device Scan" after pluging in a device to computer to get:
* MAC Address 
* Bluetooth Address 
* Internal Serial Number 
  
Supports hot swapping devices. \
Clicking on any of the Address puts the address into the Clipboard. \

For the demo I started with no devices pluged in. \
Between pressing "Device Scan" a new device was plugged in. \
